# Atividade - Hotel System

A Java-based hotel management system project created for training Object-Oriented Programming (OOP) principles.

## Description

This project implements a hotel system in Java, designed as a training exercise for learning and practicing Object-Oriented Programming concepts such as:
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Project Details

- **Language**: Java
- **Type**: Educational Project
- **Value**: 2 points (Atividade valendo 2 pontos em POO)
- **Purpose**: Training in Object-Oriented Programming

## Features

The hotel system includes functionality for managing:
- Guest reservations
- Room management
- Hotel operations

## Getting Started

1. Clone the repository
2. Compile the Java files
3. Run the application

## Learning Objectives

This project serves as a practical exercise to understand and implement core OOP principles through a real-world hotel management scenario.
